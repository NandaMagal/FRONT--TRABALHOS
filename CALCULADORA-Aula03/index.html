SUBTRAÇÃO - ESTÁ OPERAÇÃO MATEMÁTICA CALCULA A SUBTRAÇÃO ENTRE DOIS NÚMEROS INFORMADOS PELO USUÁRIO

MULTIPLICAÇÃO - ESTÁ OPERAÇÃO MATEMÁTICA CALCULA A MULTIPLICAÇÃO ENTRE DOIS NÚMEROS INFORMADOS PELO USUÁRIO

DIVISÃO - ESTÁ OPERAÇÃO MATEMÁTICA CALCULA A DIVISÃO ENTRE DOIS NÚMEROS INFORMADOS PELO USUÁRIO
